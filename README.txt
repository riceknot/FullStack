Instruction to run app:
1. Open the backend and client folders on 2 separate Visual Studio Code windows with 2 different terminals, correspond to each folder.
2. Initiate backend FIRST (write 'node server' in backend terminal).
3. Then, initiate client by writing 'npm start' in client terminal. If a message pops up and ask to run on another port than 3000, press Y.
4. Wait for client React to render is the process is finished.
